Public node bandwidth is rate-limited

bcnode.io is operated as independent COSIGO Bitcoin infrastructure. The node is public, but bandwidth, uptime, monitoring, and maintenance are not free.

To keep the node sustainable, historical block serving may be rate-limited. Stable Bitcoin peers that need higher bandwidth may request operator-verified peer review by providing their node IP address. Voluntary support helps keep the node online and may help justify higher bandwidth limits.

Support does not buy consensus influence, transaction priority, private RPC access, or guaranteed uptime.

Current daily upload cap: 5 GiB. Live percentage updates when node bandwidth data is available.
